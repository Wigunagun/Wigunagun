## 📌 Featured Project

### 🎙️ [ASR/NLP Contact Centre Intelligence](https://github.com/Wigunagun/asr-nlp-contact-centre)
> End-to-end speech-to-text and NLP pipeline for the INSW Contact Centre

Built an automated system to transcribe, classify, and summarise thousands of Indonesian-language call recordings — turning raw audio into structured reform signals for government decision-makers.

| Metric | Result |
|---|---|
| Word Error Rate (WER) | **12.6%** |
| Topic Classification (macro-F1) | **0.87** |
| Summarisation (ROUGE-1) | **0.79** |
| BERTScore | **0.83** |

**Stack:** Faster-Whisper · GPT-4.1 Mini · Apache Airflow · PostgreSQL · Streamlit
